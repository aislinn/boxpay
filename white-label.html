White label and API

boxPAY allows you to easily charge consumers to their mobile phone bill on websites and applications. For a quick integration, developers can use our self service platform, or integrate our Android SDK.

For developers who want to customize the style or brand of the payment box, easily white label it using html and CSS. For a completely unique user experience, use our API.

Getting started

Customize our standard payment box using the white label or API

  • For access to our White Label or API, contact our sales team
  • Developers, please visit here for API and Whitelabel information

White label and API facts

  • Available for all 70+ countries
  • Localized in 34 languages
  • White labeling makes it easy for you to change logos, colors, and style
  • White Label maintains the shape and content of the payment box while allowing you to customize the html and CSS
  • API allows for a 100% custom integration including content and user experience
